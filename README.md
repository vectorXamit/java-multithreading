# Java Multithreading

This repository contains Java Multithreading concepts implemented through practical examples.

## Topics Covered

### Thread Class

* Creating a thread by extending the Thread class
* Starting and executing threads

### Runnable Interface

* Creating a thread using the Runnable interface
* Separating task logic from thread management

### Thread Lifecycle

* New
* Runnable
* Running
* Waiting / Blocked
* Terminated

### Sleep Method

* Pausing thread execution for a specified time
* Handling InterruptedException

### Join Method

* Waiting for a thread to complete before continuing execution

### Synchronization

* Preventing race conditions
* Synchronizing shared resources between multiple threads

## Repository Structure

* ThreadClass.java
* RunnableInterface.java
* ThreadLifecycle.java
* SleepMethod.java
* JoinMethod.java
* SynchronizationExample.java

## Purpose

The purpose of this repository is to understand concurrent programming in Java and learn how multiple threads can execute efficiently while sharing resources safely.
